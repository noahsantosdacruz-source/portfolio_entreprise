Outils du Projet Kiosque

Toutes les applications à installer et à connaître — du développement au déploiement sur Raspberry Pi.

Oken Consulting 4 catégories · 13 outils
Catégorie 01

Coder & Gérer le code

Ton quotidien de développeur — écrire, versionner, collaborer.

VS Code

Éditeur de code

Ton éditeur principal. Écris le code du site web du kiosque, tes scripts Python et tes fichiers de configuration. Installe l'extension Remote – SSH pour coder directement dans le Raspberry Pi à distance.

Télécharger VS Code Extension Remote SSH

Git

Versionning

Le logiciel en ligne de commande pour sauvegarder l'historique de ton code. Chaque modification est tracée — tu peux revenir en arrière à tout moment.

Télécharger Git

GitHub

Dépôt central

Le site web pour héberger ton code en ligne, travailler en équipe et déclencher l'automatisation DevOps via GitHub Actions.

Ouvrir GitHub GitHub Actions
Catégorie 02

Déploiement & DevOps

Conteneuriser, automatiser et déployer sur la flotte de bornes.

Docker Desktop

Essentiel

Crée, teste et lance tes conteneurs en local sur ton PC avant de les envoyer sur le Raspberry Pi. Garantit que l'application tourne exactement de la même façon partout.

Télécharger Docker Desktop Guide débutant

Portainer

Interface Docker

Interface web visuelle installée sur le Raspberry Pi. Surveille, démarre et arrête tes conteneurs Docker en cliquant — sans ligne de commande.

Installer Portainer

GitHub Actions

CI/CD

Automatise le pipeline complet : dès que tu pousses sur GitHub, l'outil vérifie le code, fabrique l'image Docker et la déploie sur tous les Raspberry Pi de la flotte.

Voir GitHub Actions
Catégorie 03

Piloter le Raspberry Pi à distance

Contrôle la borne depuis ton PC — sans clavier ni souris branchés dessus.

Raspberry Pi Imager

Installation OS

Logiciel officiel pour flasher le système d'exploitation Linux (Raspberry Pi OS Lite) sur la carte microSD ou le SSD du Pi.

Télécharger Imager
SSH

PuTTY / Terminal

Accès SSH

Connecte-toi à distance au Raspberry Pi en ligne de commande Linux. Sur Mac/Linux, le Terminal suffit. Sur Windows, utilise PuTTY.

Télécharger PuTTY

FileZilla

Transfert SFTP

Glisse-dépose des fichiers (images, vidéos, configs) depuis ton PC directement dans le Raspberry Pi à travers le réseau — comme un dossier Windows classique.

Télécharger FileZilla

VNC Viewer

Bureau à distance

Vois l'écran du Raspberry Pi en temps réel sur ton PC pour vérifier que l'affichage du kiosque est correct, sans avoir à brancher un moniteur.

Télécharger VNC Viewer Alternative : AnyDesk
Catégorie 04

Système Kiosque sur le Pi

Les applications internes à Linux qui font tourner la borne en production.

FullPageOS

OS Kiosque

Version modifiée de Linux conçue exprès pour les kiosques. Démarre directement sur Chromium en plein écran sans configuration — zéro bureau graphique inutile.

Voir sur GitHub

Chromium

Navigateur kiosque

La version open-source de Chrome. Tourne en arrière-plan sur la borne pour afficher le site web publicitaire en mode --kiosk — plein écran, sans barre d'adresse.

En savoir plus
CLI

Unclutter

Utilitaire Linux

Mini-application Linux qui cache automatiquement le curseur de la souris après 1–2 secondes d'inactivité. Indispensable pour les écrans tactiles de kiosque.

Package Debian
