Catégorie 05

Commandes Linux essentielles

Les lignes de commande à connaître pour configurer, sécuriser et piloter le kiosque.

Installation des paquets
sudo apt update && sudo apt upgrade -y Met à jour la liste des paquets et installe les mises à jour
sudo apt install --no-install-recommends xserver-xorg x11-xserver-utils xinit openbox chromium-browser Installe le serveur d'affichage léger + Chromium pour le mode kiosque
sudo apt install unclutter Installe Unclutter pour cacher le curseur automatiquement
sudo apt install docker.io Installe Docker sur le Raspberry Pi
Configuration du mode kiosque
xset s off Désactive l'économiseur d'écran
xset s noblank Empêche l'écran de s'éteindre
xset -dpms Désactive la gestion d'énergie de l'écran
unclutter -idle 0.1 -root & Cache le curseur après 0.1 seconde d'inactivité
chromium-browser --noerrdialogs --disable-infobars --kiosk https://votre-url.com & Lance Chromium en plein écran sur l'URL du kiosque
Docker
docker build -t mon-kiosque . Construit l'image Docker depuis le Dockerfile
docker run -d --restart always -p 80:80 mon-kiosque Lance le conteneur en arrière-plan, redémarre automatiquement
docker ps Liste les conteneurs en cours d'exécution
docker logs mon-kiosque Affiche les logs du conteneur
docker pull ghcr.io/noahsantosdacruz-source/kiosque:latest Télécharge la dernière image depuis GitHub Container Registry
Sécurité & robustesse
sudo raspi-config Ouvre l'utilitaire de configuration du Pi (activer Overlay FS, SSH…)
sudo systemctl enable watchdog Active le watchdog — redémarre le Pi automatiquement en cas de freeze
sudo chmod 700 ~/.ssh Sécurise le dossier de clés SSH
sudo passwd pi Change le mot de passe de l'utilisateur pi (obligatoire en prod)
SSH & accès à distance
ssh pi@192.168.x.x Se connecte au Raspberry Pi via SSH (remplace l'IP)
ssh-keygen -t ed25519 Génère une paire de clés SSH sécurisée
ssh-copy-id pi@192.168.x.x Copie ta clé publique sur le Pi pour se connecter sans mot de passe
scp fichier.html pi@192.168.x.x:/home/pi/ Transfère un fichier depuis ton PC vers le Pi
Surveillance système
htop Affiche l'utilisation CPU / RAM en temps réel
vcgencmd measure_temp Affiche la température du processeur du Raspberry Pi
df -h Affiche l'espace disque disponible
sudo journalctl -f Affiche les logs système en temps réel
sudo reboot Redémarre proprement le Raspberry Pi
